Description:
Five different logical games with instructions inside.

For install - clone the repository:
    
    git clone git@github.com:Mirdalan-p/python-project-49.git
 
Use Makefile:
 
    make install
    
    make build
    
    make publish
    
    make package-install

To start game just type its name:
